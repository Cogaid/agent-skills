## Receipt Checklist

```markdown
# Receipt Documentation Checklist

For each expense, verify:

## Required Information
- [ ] Vendor/merchant name
- [ ] Date of purchase
- [ ] Total amount (including tax)
- [ ] Payment method visible

## For Meals ($75+)
- [ ] Itemized list of items
- [ ] Names of all attendees
- [ ] Business purpose documented
- [ ] Company/organization of guests

## For Travel
- [ ] Destination noted
- [ ] Dates of travel
- [ ] Business purpose
- [ ] Pre-approval reference

## Receipt Quality
- [ ] Clearly legible
- [ ] Complete (not cut off)
- [ ] Original or clear copy
- [ ] Properly attached to report

## Missing Receipt Protocol
If receipt is missing:
1. [ ] Attempt to get duplicate from vendor
2. [ ] Document bank/card statement
3. [ ] Write detailed memo
4. [ ] Obtain manager approval
5. [ ] Note exception in report
```
