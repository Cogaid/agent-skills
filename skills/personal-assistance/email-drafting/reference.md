## Subject Line Best Practices

### Do:
- Be specific: "Q3 Budget Approval Needed by Friday"
- Front-load key words: "ACTION REQUIRED: Contract Review"
- Include date if time-sensitive: "Meeting Agenda - March 15"
- Use brackets for scanning: "[Feedback Request] New Logo Designs"

### Don't:
- Be vague: "Quick question"
- ALL CAPS entire subject
- Use "URGENT" unless truly urgent
- Write a sentence as a subject line

### Subject Line Formulas

| Purpose | Format | Example |
|---------|--------|---------|
| Request | [Request]: [Specific ask] | "Request: Headcount data for Q3 planning" |
| FYI | [FYI]: [Topic] | "FYI: Office closed Monday for holiday" |
| Action | ACTION: [Task] by [Date] | "ACTION: Review proposal by EOD Friday" |
| Follow-up | Follow-up: [Original topic] | "Follow-up: Marketing partnership discussion" |
| Introduction | Intro: [Name] <> [Name] | "Intro: Sarah Chen <> David Park" |
| Decision | DECISION: [What] | "DECISION: Vendor selection for Q4" |
| Update | Update: [Project/Topic] | "Update: Website redesign progress" |

## Tone Guide

### Formal (External, executives, first contact)
- Full sentences, complete greetings
- "Dear [Name]" or "Hello [Name]"
- "Best regards" or "Sincerely"
- Avoid contractions
- No emoji or casual language

**Example**:
```
Dear Mr. Thompson,

I hope this message finds you well. I am writing to follow up on
our discussion regarding the partnership opportunity.

Best regards,
[Name]
```

### Professional (Default for most work email)
- "Hi [Name]"
- Clear and direct but friendly
- "Thanks" or "Best"
- Contractions okay
- Focus on clarity

**Example**:
```
Hi Sarah,

Following up on our budget discussion. I've attached the revised
numbers. Let me know if you have questions.

Thanks,
[Name]
```

### Casual (Close colleagues, established relationships)
- "Hey [Name]"
- Conversational tone
- "Thanks!" or "Cheers"
- Can use appropriate humor
- Keep it brief

**Example**:
```
Hey Marcus,

Quick heads up - the client moved the meeting to 3pm.
Updated invite incoming.

Cheers
```

## Common Fixes

### Before: Burying the lead
```
I wanted to reach out because we had a great meeting last week
and I've been thinking about what you said regarding the project
timeline. After reviewing everything, I think we should...
```

### After: Leading with the point
```
I recommend we extend the project deadline by two weeks.

Here's why: [reasons]
```

---

### Before: Vague request
```
Can you help me with the report?
```

### After: Specific request
```
Could you review pages 5-10 of the report and confirm the revenue
figures by Thursday?
```

---

### Before: Wall of text
```
[Long paragraph with multiple topics, requests, and information
all jumbled together making it hard to parse what's important
and what action is needed...]
```

### After: Scannable structure
```
Summary: [One line]

Context:
- [Bullet 1]
- [Bullet 2]

Request:
[Specific ask with deadline]
```

---

### Before: Wishy-washy
```
I was wondering if maybe you might have time to possibly look at
this when you get a chance, if it's not too much trouble...
```

### After: Direct
```
Could you review this by Friday?
```

## Editing Checklist

### Before Sending

**Content**:
- [ ] Subject line is specific and clear
- [ ] Purpose stated in first 2 sentences
- [ ] Only necessary context included
- [ ] Request/action is specific
- [ ] Deadline stated (if applicable)
- [ ] Next steps are clear

**Tone**:
- [ ] Tone matches relationship
- [ ] No unnecessary hedging
- [ ] Not too casual or too formal

**Mechanics**:
- [ ] Proofread for typos
- [ ] Names spelled correctly
- [ ] Right recipients (To vs CC vs BCC)
- [ ] Attachments actually attached
- [ ] Links work

## Shortening Emails

Most emails can be cut by 50%. Look for:

### Remove:
- "I hope this email finds you well"
- "I just wanted to reach out"
- "Per my last email"
- "As discussed"
- "Please don't hesitate to"
- "I wanted to follow up regarding"

### Replace:
| Wordy | Concise |
|-------|---------|
| "I am writing to inform you that" | [Just state it] |
| "at this point in time" | "now" |
| "in order to" | "to" |
| "due to the fact that" | "because" |
| "I was wondering if you could" | "Could you" |
| "Please find attached" | "Attached is" |
| "I would like to" | "I want to" |
| "If you have any questions" | [Optional - often unnecessary] |

### Word Count Targets
- Quick request: 50-100 words
- Update/FYI: 100-150 words
- Detailed request: 150-250 words
- Rarely exceed: 300 words

## Email Response Times

### Guidelines
| Type | Target Response |
|------|-----------------|
| Urgent (boss, client, crisis) | <2 hours |
| Normal work request | <24 hours |
| Low priority/FYI | <48 hours |
| Can wait | Weekly batch |

### If You Can't Respond Fully
```
Hi [Name],

Got your email. I'll send a full response by [day].

[Name]
```
