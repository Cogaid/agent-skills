# Reschedule & Cancellation Templates

## Reschedule Request

### Standard Reschedule

```
Subject: Reschedule Request: [Meeting Name]

Hi [Name(s)],

I need to reschedule our [meeting name] originally set for
[Day, Date at Time].

Reason: [Brief, honest explanation]

Alternative times that work for me:
1. [Day, Date] at [Time] [Timezone]
2. [Day, Date] at [Time] [Timezone]
3. [Day, Date] at [Time] [Timezone]

Please let me know which works best, or suggest alternatives.

Apologies for any inconvenience.

Thanks,
[Name]
```

---

### Last-Minute Reschedule

```
Subject: URGENT: Need to reschedule [Meeting Name] today

Hi [Name],

I'm very sorry - I need to reschedule our [time] meeting today.

[Brief reason: family emergency / urgent work matter / etc.]

Can we move to one of these times instead?
- [Option 1]
- [Option 2]
- [Option 3]

If none work, I can make any time [tomorrow/next day] work.

Again, apologies for the short notice.

[Name]
```

---

### Reschedule Due to Conflict

```
Subject: Reschedule Request: [Meeting Name] - Conflict

Hi [Name],

A conflict has come up for our [meeting time] slot - [brief
explanation: exec meeting moved, client emergency, etc.].

I want to make sure we still connect. Would any of these work?
- [Option 1]
- [Option 2]
- [Option 3]

If the original agenda isn't time-sensitive, we could also
push to next week: [suggest time].

Let me know what works best.

Thanks for understanding,
[Name]
```

## Cancellation Messages

### Meeting No Longer Needed

```
Subject: Cancelled: [Meeting Name]

Hi [Name(s)],

I'm cancelling our [meeting name] scheduled for [date/time].

Reason: [We resolved this via email / decision was made /
no longer relevant]

No need to reschedule - I'll reach out if anything changes.

Thanks,
[Name]
```

---

### Cancellation with Reschedule

```
Subject: Cancelled + Reschedule: [Meeting Name]

Hi [Name(s)],

I need to cancel our [meeting name] on [date/time].

Reason: [Explanation]

I'll send a new invite for [proposed new time] shortly.
Please let me know if that doesn't work.

Thanks,
[Name]
```

---

### Cancellation Due to Low Priority

```
Subject: Cancelled: [Meeting Name] - Let's Go Async

Hi [Name(s)],

I'm cancelling our [meeting] scheduled for [date/time].

After reviewing the agenda, I think we can handle this
async instead. Here's what I propose:

- [Action 1]: [Owner] adds thoughts to [doc/thread]
- [Action 2]: [Owner] reviews by [date]
- [Action 3]: We reconvene only if needed

Please free up this time on your calendar. Let me know if
you disagree and prefer to keep the meeting.

Thanks,
[Name]
```

---

### Recurring Meeting Cancellation (One Instance)

```
Subject: Skip: [Meeting Name] - [Date]

Hi team,

Let's skip [meeting name] this [day] ([date]).

Reason: [Holiday / low attendance / no urgent items]

Our next session will be [next occurrence date].

If you have anything urgent, reach out directly or add to
the doc for next time.

Thanks,
[Name]
```

---

### End Recurring Meeting Series

```
Subject: Ending: [Meeting Name] Recurring Series

Hi team,

I'm ending our recurring [meeting name] effective [date].

Reason: [Purpose is complete / team restructured / going
async / frequency changing]

What's next:
- [What replaces this, if anything]
- [How to raise topics that would have come up here]

Thanks for making these meetings valuable. Feel free to
reach out with questions.

[Name]
```

## Decline Meeting Templates

### Decline - Not Required

```
Hi [Organizer],

Thanks for the invite to [meeting].

I don't think I need to be in this one - [brief reason:
not involved in this project / my input isn't needed / etc.]

Would it work if I:
- Got the notes afterward?
- Provided input async before?

Let me know if I'm missing something.

Thanks,
[Name]
```

---

### Decline - Conflict

```
Hi [Organizer],

Thanks for the invite. Unfortunately I have a conflict at
that time with [type of conflict: another meeting / blocked
focus time].

A few options:
1. Send me the notes and I'll provide input async
2. I could join just for [specific part] at [time]
3. Let me know if you reschedule to [alternative times]

Let me know what works best.

[Name]
```

---

### Decline - Suggest Delegate

```
Hi [Organizer],

Thanks for including me. For this topic, [colleague name]
would be better suited - they're closer to [reason].

I've cc'd them here. [Colleague], would you be able to
cover this?

Let me know if you specifically need me there for [reason].

Thanks,
[Name]
```

---

## Tips for Rescheduling/Cancelling

1. **Give notice**: As much as possible
2. **Be honest**: Brief reason helps, excuses don't
3. **Offer alternatives**: Don't make them chase you
4. **Apologize once**: Don't over-apologize
5. **Update the calendar**: Don't leave ghost meetings
6. **Consider the impact**: Cancel gracefully with important attendees
