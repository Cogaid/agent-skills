## Time Zone Management

### Common Time Zones

| Abbreviation | Full Name | UTC Offset |
|--------------|-----------|------------|
| PT | Pacific Time | UTC-8 (PST) / UTC-7 (PDT) |
| MT | Mountain Time | UTC-7 (MST) / UTC-6 (MDT) |
| CT | Central Time | UTC-6 (CST) / UTC-5 (CDT) |
| ET | Eastern Time | UTC-5 (EST) / UTC-4 (EDT) |
| GMT/UTC | Greenwich/Coordinated | UTC+0 |
| CET | Central European | UTC+1 (CET) / UTC+2 (CEST) |
| IST | India Standard | UTC+5:30 |
| CST (China) | China Standard | UTC+8 |
| JST | Japan Standard | UTC+9 |
| AEST | Australian Eastern | UTC+10 / UTC+11 |

### Global Team Overlap Windows

| Team Locations | Best Overlap (Local Times) |
|----------------|---------------------------|
| US West + US East | 9am-5pm PT / 12pm-8pm ET |
| US + Europe | 8-11am ET / 2-5pm CET |
| US + India | 8-10am ET / 5:30-7:30pm IST |
| US West + APAC | 5-8pm PT / 9am-12pm JST+1 day |
| Europe + India | 12-3pm CET / 4:30-7:30pm IST |
| Europe + APAC | 8-10am CET / 4-6pm JST |

### Time Zone Best Practices

1. **Always specify the time zone** when proposing times
2. **Use recipient's time zone** in your message to them
3. **Rotate inconvenient times** fairly across global teams
4. **Account for Daylight Saving** - transitions differ by country
5. **Use scheduling tools** like Calendly or World Time Buddy

## Meeting Etiquette

### For the Organizer

**Before the Meeting**:
- Send agenda at least 24 hours ahead
- Include pre-work with clear deadlines
- Confirm attendees and video links work
- Prepare materials and talking points

**During the Meeting**:
- Start on time (don't wait for stragglers)
- State the purpose at the start
- Assign a note-taker
- Keep discussion on track
- End 5 minutes early for buffer

**After the Meeting**:
- Share notes within 24 hours
- Document decisions and action items
- Follow up on assigned tasks
- Schedule any needed follow-up meetings

### For Attendees

- Join 1-2 minutes early (especially video)
- Complete pre-work before meeting
- Come prepared to contribute
- Stay present (no email/Slack during)
- Mute when not speaking (video calls)
- Speak up if discussion goes off track

## Recurring Meeting Management

### Setting Up Recurring Meetings

Include in the invite:
- Clear purpose statement
- Standing agenda template
- End date or review date (not indefinite)
- Cancellation policy (skip if no agenda)

### Quarterly Recurring Meeting Audit

For each recurring meeting, ask:
- [ ] Is this meeting still necessary?
- [ ] Are the right people attending?
- [ ] Is the frequency correct? (Weekly → Biweekly?)
- [ ] Are we achieving the stated purpose?
- [ ] Can it be shorter?
- [ ] Should it be async instead?

### Declining Unnecessary Meetings

```
Hi [Organizer],

Thanks for including me in [meeting name].

After reviewing the agenda, I don't think I'm essential for
this discussion. Would it work if I:
- Received the notes instead?
- Provided input async beforehand?
- Joined only for [specific agenda item]?

Let me know if there's a specific reason you need me there.

Thanks,
[Name]
```

## Meeting-Free Time Protection

### Focus Time Strategies

- Block 2-4 hour chunks for deep work
- Mark focus blocks as "Busy" not "Free"
- Set calendar to auto-decline during focus time
- Communicate focus hours to team

### Meeting Days Strategy

- Cluster meetings on 2-3 days (e.g., Tues/Wed/Thurs)
- Keep Monday/Friday for focused work
- Batch external meetings on specific days
- Protect mornings for creative work

### Calendar Hygiene

- Review calendar Sunday night or Monday morning
- Decline/reschedule non-essential meetings
- Consolidate back-to-back meetings
- Add buffer between intensive meetings
