## Research Methodologies

### Quick Research (15-30 minutes)

**Best for:**
- Simple factual questions
- Quick background checks
- Preliminary exploration

**Process:**
1. Search 2-3 reliable sources
2. Cross-check key facts
3. Note source quality
4. Summarize findings
5. List sources

### Standard Research (1-2 hours)

**Best for:**
- Moderate complexity questions
- Decision support
- Comparison analysis

**Process:**
1. Define research question clearly
2. Identify 5-10 potential sources
3. Review and extract relevant info
4. Organize findings by theme
5. Analyze and synthesize
6. Write summary with citations

### Deep Research (4-8 hours)

**Best for:**
- Complex topics
- High-stakes decisions
- Comprehensive understanding

**Process:**
1. Create research plan
2. Map information landscape
3. Systematic source review (15+)
4. Document everything
5. Identify patterns and gaps
6. Multi-perspective analysis
7. Detailed report with recommendations

## Source Evaluation

### The CRAAP Test

| Criteria | Questions to Ask |
|----------|------------------|
| **C**urrency | When was it published? Updated? |
| **R**elevance | Does it relate to your question? |
| **A**uthority | Who is the author? Credentials? |
| **A**ccuracy | Is info supported by evidence? |
| **P**urpose | Why does this exist? Bias? |

### Source Reliability Tiers

**Tier 1: Highly Reliable**
- Peer-reviewed journals
- Government statistics
- Official company filings
- Established news organizations
- Academic institutions

**Tier 2: Generally Reliable**
- Industry publications
- Expert interviews
- Reputable news sites
- Research firms
- Professional organizations

**Tier 3: Use with Caution**
- Blog posts
- Social media
- Wikipedia (verify citations)
- Opinion pieces
- User reviews

**Tier 4: Verification Required**
- Anonymous sources
- Unattributed claims
- Outdated content
- Known biased sources
- Single-source claims

### Red Flags

- No author attribution
- No publication date
- No citations or references
- Emotional language
- One-sided presentation
- Factual errors
- Broken links
- Poor grammar/presentation

## Information Synthesis

### Organizing Findings

**By Theme:**
```
Theme 1: [Topic Area]
  - Finding A (Source 1)
  - Finding B (Source 2, 3)
  - Conflicting: Finding C vs D

Theme 2: [Topic Area]
  - Finding E (Source 4)
  ...
```

**By Source:**
```
Source 1: [Name, Date]
  - Key point A
  - Key point B
  - Reliability: [Assessment]

Source 2: [Name, Date]
  ...
```

**By Question:**
```
Question 1: [Research Question]
  Answer: [Synthesis of findings]
  Sources: [List]
  Confidence: [High/Medium/Low]

Question 2: ...
```

### Handling Conflicting Information

1. **Note the conflict** - Document both perspectives
2. **Check sources** - Which is more reliable?
3. **Look for context** - Different time periods? Definitions?
4. **Seek additional sources** - What do others say?
5. **Present both** - If unresolved, show both views

### Drawing Conclusions

**Strong conclusions require:**
- Multiple supporting sources
- No significant contradictions
- Reliable primary sources
- Recent/current information
- Expert consensus

**Weak conclusions (acknowledge):**
- Limited sources available
- Conflicting information
- Outdated data
- Single source only
- Expert disagreement

## Bias Awareness

### Types of Bias to Watch

**Source Bias:**
- Corporate interests
- Political affiliation
- Funding sources
- Geographic perspective
- Industry involvement

**Research Bias:**
- Confirmation bias (seeking supporting info)
- Selection bias (cherry-picking sources)
- Recency bias (overweighting new info)
- Authority bias (assuming experts are right)

**Presentation Bias:**
- Framing effects
- Omission of contradicting info
- Statistical manipulation
- Misleading visualizations

### Mitigation Strategies

1. **Seek diverse sources** - Different perspectives, regions, types
2. **Question assumptions** - Why do I believe this?
3. **Look for contrary evidence** - Actively seek opposing views
4. **Check funding/affiliations** - Who benefits from this info?
5. **Document your process** - Be transparent about methods

## Citation Guidelines

### Basic Citation Format

```
[Author Last Name, First Name]. "[Title]." [Publication],
[Date Published]. [URL]. Accessed [Date Accessed].
```

### Quick Reference Format

For internal research notes:
```
[Source Name] ([Date]) - [Key Point] - [URL]
```

### Source Tracking Template

```
Source ID: [Unique identifier]
Title: [Full title]
Author: [Name and credentials]
Publication: [Where published]
Date: [Publication date]
URL: [Full URL]
Access Date: [When you accessed it]
Reliability: [Tier 1-4]
Key Findings:
  - [Finding 1]
  - [Finding 2]
Notes: [Any caveats or observations]
```

## Report Structures

### Executive Summary Format

```
Research Question:
[Clear statement of what was researched]

Key Findings:
1. [Most important finding]
2. [Second most important]
3. [Third most important]

Recommendation:
[Clear actionable recommendation]

Confidence Level: [High/Medium/Low]
Based on: [X sources reviewed]
```

### Detailed Report Format

```
1. Executive Summary
   - Research objective
   - Key findings
   - Recommendations

2. Methodology
   - Research approach
   - Sources consulted
   - Limitations

3. Findings
   - Section by topic
   - Supporting evidence
   - Source citations

4. Analysis
   - Patterns identified
   - Implications
   - Uncertainties

5. Recommendations
   - Action items
   - Priorities
   - Next steps

6. Appendices
   - Source list
   - Raw data
   - Additional details
```

### Comparison Matrix Format

```
| Criteria | Option A | Option B | Option C |
|----------|----------|----------|----------|
| [Criterion 1] | [Rating] | [Rating] | [Rating] |
| [Criterion 2] | [Rating] | [Rating] | [Rating] |
| ...      |          |          |          |
| **Score** | [Total] | [Total] | [Total] |

Recommendation: [Option] because [reasoning]
Sources: [List]
```
