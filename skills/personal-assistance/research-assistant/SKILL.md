## Purpose

Conduct comprehensive research on topics, compile findings systematically, and present actionable insights for informed decision-making.

## Quick Reference

### Research Types

| Type | Duration | Depth | Output |
|------|----------|-------|--------|
| Quick Scan | 15-30 min | Surface | Key facts + sources |
| Standard | 1-2 hours | Moderate | Summary + analysis |
| Deep Dive | 4-8 hours | Comprehensive | Full report |
| Ongoing | Days/Weeks | Thorough | Multi-part report |

### Research Quality Indicators

| Element | Poor | Good | Excellent |
|---------|------|------|-----------|
| Sources | 1-2 | 5-10 | 15+ diverse |
| Recency | Any | < 2 years | < 6 months |
| Variety | Single type | 2-3 types | Multiple types |
| Verification | Unchecked | Spot-checked | Cross-verified |

## Research Framework

### The GATHER Method

| Step | Action | Purpose |
|------|--------|---------|
| **G**oal | Define objective | Clear research question |
| **A**ssess | Evaluate scope | Time and depth needed |
| **T**rack | Document sources | Maintain credibility |
| **H**arvest | Collect data | Systematic information gathering |
| **E**valuate | Analyze findings | Critical assessment |
| **R**eport | Synthesize results | Actionable output |

## Workflow Checklist

### 1. Research Planning
- [ ] Clarify research question
- [ ] Define success criteria
- [ ] Set time/depth parameters
- [ ] Identify key sources
- [ ] Create research outline

### 2. Information Gathering
- [ ] Search primary sources
- [ ] Review secondary sources
- [ ] Document all sources
- [ ] Note conflicting information
- [ ] Identify knowledge gaps

### 3. Source Evaluation
- [ ] Check source credibility
- [ ] Verify information currency
- [ ] Cross-reference facts
- [ ] Note potential biases
- [ ] Assess reliability

### 4. Analysis
- [ ] Organize findings
- [ ] Identify patterns
- [ ] Draw conclusions
- [ ] Note limitations
- [ ] Formulate recommendations

### 5. Reporting
- [ ] Structure findings
- [ ] Cite all sources
- [ ] Highlight key insights
- [ ] Provide recommendations
- [ ] Include next steps
