## Long Trip (7+ days)

### Strategies
- **Do laundry mid-trip** vs packing everything
- **Ship items ahead** if space is tight
- **Pack layers** for temperature changes
- **Compression bags** for bulky items

### Additional Items
- [ ] Laundry bag/detergent sheets
- [ ] More medications (original containers)
- [ ] Backup credit card (different location)
- [ ] Multi-day itinerary printed

## Carry-On Only Guide

### Bag Specs
- Size: Under 22" x 14" x 9"
- Plus: 1 personal item (backpack/laptop bag)

### Packing System
```
CARRY-ON LAYOUT:
├── Packing Cube 1: Shirts/tops (rolled)
├── Packing Cube 2: Pants/bottoms (folded)
├── Packing Cube 3: Undergarments/socks
├── Toiletry bag: TSA 3-1-1 compliant
├── Shoes: In shoe bags at bottom
└── Tech/docs: Accessible pocket
```

### Wear on Plane
- Bulkiest shoes
- Jacket/blazer
- Heaviest pants

### TSA 3-1-1 Rule
- 3.4 oz (100ml) containers max
- 1 quart-sized clear bag
- 1 bag per passenger

## Quick Reference by Trip Length

### 1-2 Days
- Carry-on only
- 2 outfits + what you wear
- Minimal toiletries

### 3-4 Days
- Carry-on only (still possible)
- 3-4 outfits + wear
- Full toiletry kit

### 5-7 Days
- Carry-on or checked
- 5 outfits + wear + backup
- Consider mid-trip laundry

### 7+ Days
- Checked bag likely
- Plan for laundry
- Pack versatile pieces

## Post-Trip Reset

### Immediate (Day of Return)
- [ ] Unpack completely
- [ ] Start laundry
- [ ] Charge all devices
- [ ] File receipts for expenses

### Within 24 Hours
- [ ] Submit expense report
- [ ] Send follow-up emails
- [ ] Update CRM/notes
- [ ] Restock travel toiletries

### Within Week
- [ ] Process any business cards
- [ ] Complete trip report if needed
- [ ] Schedule any follow-up meetings
