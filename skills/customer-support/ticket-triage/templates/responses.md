## Bug Acknowledgment

```
Subject: Re: [Original Subject] - We're investigating [Ticket #]

Hi [Name],

Thank you for reporting this issue. I understand [brief restatement of problem],
and I can see how this is impacting your workflow.

To help us investigate, could you please provide:
1. Steps to reproduce the issue
2. Browser/device you're using
3. Approximate time when this started
4. Any error messages you're seeing (screenshots help!)

I've flagged this with our team and will keep you updated on progress.

Best,
[Agent Name]
```

## Access Issues Response

```
Subject: Re: [Original Subject] [Ticket #]

Hi [Name],

I understand you're having trouble accessing your account - let's get this sorted.

First, let's try a few things:
1. Clear your browser cache and cookies
2. Try an incognito/private window
3. Ensure you're using the correct email: [masked email if available]

If those don't work, I can help with:
- Password reset: [Reset link]
- Account verification
- SSO configuration (if applicable)

Reply back with what you've tried and any error messages you're seeing.

Best,
[Agent Name]
```

## Performance Response

```
Subject: Re: [Original Subject] [Ticket #]

Hi [Name],

I'm sorry to hear you're experiencing slow performance - that's definitely
not the experience we want you to have.

A few quick questions to help us investigate:
1. When did this start happening?
2. Is it affecting specific features or the entire application?
3. What browser/device are you using?
4. Are other users on your team experiencing the same issue?

In the meantime, you might try:
- Clearing your browser cache
- Trying a different browser
- Checking your internet connection

I'll keep you posted as we look into this.

Best,
[Agent Name]
```
