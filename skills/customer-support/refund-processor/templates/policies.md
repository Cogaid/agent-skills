# Refund Policies Reference

## Standard Return Policy

### Overview

| Attribute | Value |
|-----------|-------|
| Return Window | 30 days from delivery |
| Condition Required | Original, unused, with tags |
| Receipt Required | Yes (order number or email) |
| Restocking Fee | Varies by category |
| Return Shipping | Customer paid (unless our error) |

### Category-Specific Policies

#### Electronics
```
Window: 15 days
Condition: Unopened preferred, opened accepted
Restocking Fee: 15% if opened
Exceptions: Defective items = full refund + free shipping
```

#### Apparel & Accessories
```
Window: 30 days
Condition: Tags attached, unworn, unwashed
Restocking Fee: None
Exceptions: Intimate apparel, swimwear = final sale
```

#### Furniture & Large Items
```
Window: 30 days
Condition: Original packaging, unassembled
Restocking Fee: 20%
Return Shipping: Customer arranges + pays
Exceptions: Defective = we arrange + pay
```

#### Digital Products
```
Window: None (non-refundable once accessed)
Exceptions:
  - Technical issues preventing access
  - Duplicate purchase
  - Unauthorized purchase
```

#### Subscription Services
```
Window: Within billing cycle
Refund Type: Prorated
Auto-Renewal: Can cancel anytime for future
Exceptions: Annual plans may have different terms
```

#### Custom/Personalized Items
```
Window: None (non-refundable)
Exceptions:
  - Manufacturing defect
  - Error in customization (our fault)
  - Significant quality issues
```

## Refund Types

### 1. Full Refund
**Includes:**
- Original purchase price
- Original shipping (if our error)
- Applicable taxes

**Granted When:**
- Item defective/damaged
- Wrong item sent
- Item never received
- Within return window + meets conditions

### 2. Partial Refund
**Calculated As:**
- Purchase price minus deductions

**Deductions May Include:**
- Restocking fee
- Missing components
- Wear/damage assessment
- Original shipping (if not our error)

**Granted When:**
- Open-box returns
- Missing parts/accessories
- Minor damage
- Outside policy exception

### 3. Store Credit
**Features:**
- Usually 10% bonus over cash refund
- No expiration (or 12 months)
- Transferable (or not)

**Offered When:**
- Customer preference
- Faster resolution
- Outside return window (as exception)
- Gift returns without receipt

### 4. Exchange
**Options:**
- Same item, different size/color
- Different item of equal value
- Different item with price adjustment

**Processing:**
- Ship new item when return received, OR
- Ship new item immediately with return label

## Exception Guidelines

### Factors to Consider

#### Customer Value Score

| Level | Criteria | Exception Flexibility |
|-------|----------|----------------------|
| VIP | $5,000+ lifetime | High |
| Gold | $1,000+ lifetime | Medium |
| Silver | $500+ lifetime | Low |
| New | < $500 | Very Low |

#### Request History

| History | Recommendation |
|---------|----------------|
| First ever request | Lean toward approval |
| 1-2 requests/year | Standard policy |
| 3+ requests/year | Strict policy |
| Pattern of abuse | Flag for review |

#### Dollar Amount

| Amount | Approval Level |
|--------|---------------|
| < $25 | Agent discretion |
| $25 - $100 | Agent with justification |
| $100 - $500 | Team lead approval |
| > $500 | Manager approval |

### Exception Documentation

**Required Fields:**

```
6. Previous Exceptions (12 mo):
7. Justification:
8. Approved By:
9. Date:
```

## Shipping Policies for Returns

### Who Pays?

| Scenario | Pays For Return |
|----------|-----------------|
| Changed mind | Customer |
| Wrong size ordered | Customer |
| Defective item | Company |
| Wrong item sent | Company |
| Damaged in shipping | Company |
| Not as described | Company |

### Return Label Options

**Prepaid Label (Company Paid):**
- Deducted from refund, OR
- Free for qualifying returns

**Customer-Arranged:**
- Customer chooses carrier
- Tracking required
- Insurance recommended

### International Returns

```
Responsibility: Customer pays shipping
Customs: Customer handles
Timeline: Allow 2-4 weeks
Refund: Minus original shipping costs
Exceptions: Major defects only
```

## Fraud Prevention Policies

### Automatic Flags

System flags account when:
- 3+ refunds in 90 days
- Refund rate > 50% of orders
- Claims "never received" 2+ times
- Disputes refund denials with bank

### Required Verification

**For High-Risk Refunds:**
- Photo of defect
- Return of item before refund
- Signed declaration
- Manager approval

### Blacklist Criteria

**Account may be restricted when:**
- Confirmed fraud
- Chargeback abuse
- False claims proven
- Policy violations

## Regulatory Compliance

### Credit Card Rules

**Visa/Mastercard:**
- Must refund to original card
- Cannot require store credit only
- 30-day processing requirement

### Consumer Protection Laws

**Know Your Region:**
- EU: 14-day cooling off period
- California: 7-day presumed receipt
- Australia: Consumer guarantees

### Documentation Retention

|---------------|------------------|

## Process Flowcharts

### Standard Refund Flow

```
[Request Received]
       ↓
[Verify Order] → [Not Found] → [Request Details]
       ↓
[Check Window] → [Outside] → [Exception Review]
       ↓
[Check Condition] → [Issues] → [Partial Refund Calc]
       ↓
[Calculate Amount]
       ↓
[Check Approval Level]
       ↓
[Process Refund]
       ↓
[Notify Customer]
       ↓
[Close Case]
```

### Exception Request Flow

```
[Standard Denial]
       ↓
[Customer Appeals]
       ↓
[Gather Customer Value Data]
       ↓
[Assess Exception Factors]
       ↓
[Document Justification]
       ↓
[Route to Appropriate Approver]
       ↓
[Decision Made]
       ↓
[Document Outcome]
       ↓
[Communicate to Customer]
```
