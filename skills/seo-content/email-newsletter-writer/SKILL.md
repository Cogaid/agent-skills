## Purpose

Write compelling email newsletters that engage subscribers, deliver value, and drive desired actions.

## Quick Reference

### Email Types

| Type | Frequency | Purpose | Length |
|------|-----------|---------|--------|
| Weekly Digest | Weekly | Curated content | Medium |
| Product Update | Monthly | New features | Short |
| Educational | Bi-weekly | How-to content | Long |
| Promotional | As needed | Sales/offers | Short |
| Personal | Weekly | Connection | Medium |

### Performance Benchmarks

| Metric | Average | Good | Great |
|--------|---------|------|-------|
| Open Rate | 20% | 25%+ | 35%+ |
| Click Rate | 2.5% | 4%+ | 6%+ |
| Unsubscribe | 0.5% | <0.3% | <0.1% |

## Newsletter Structure

### Essential Elements

| Element | Purpose | Position |
|---------|---------|----------|
| Preheader | Expand subject line | First line |
| Hook | Grab attention | Opening |
| Value | Deliver promise | Body |
| CTA | Drive action | Throughout/End |
| P.S. | Second chance | Footer |

## Workflow Checklist

### 1. Planning
- [ ] Define newsletter goal
- [ ] Identify target audience
- [ ] Choose primary CTA
- [ ] Gather content/links
- [ ] Set send time

### 2. Subject Line
- [ ] Write 5-10 options
- [ ] Keep under 50 chars
- [ ] Create curiosity/value
- [ ] A/B test top 2
- [ ] Write preheader

### 3. Body Copy
- [ ] Strong opening hook
- [ ] Deliver value promised
- [ ] Use subheadings
- [ ] Include visuals
- [ ] Clear CTAs

### 4. Review
- [ ] Proofread thoroughly
- [ ] Test all links
- [ ] Check personalization
- [ ] Mobile preview
- [ ] Spam check

### 5. Send
- [ ] Segment audience
- [ ] Schedule optimal time
- [ ] Monitor deliverability
- [ ] Track opens/clicks
- [ ] Note learnings

## Subject Line Formulas

### High-Converting Formats

```
[Number] + [Topic] + [Benefit]
→ "5 ways to double your productivity today"

[Question] + [Curiosity]
→ "Are you making this common mistake?"

[How to] + [Desired Outcome]
→ "How to write emails that convert"

[Personalization] + [Relevance]
→ "[Name], your weekly marketing insights"

[Urgency] + [Value]
→ "Last chance: 40% off ends tonight"
```

## Best Practices

### Do's
- Deliver value in every email
- Use conversational tone
- Include one clear CTA
- Optimize for mobile
- Segment your list

### Don'ts
- Use ALL CAPS in subject
- Overuse exclamation points
- Send without testing
- Ignore unsubscribes
- Use misleading subjects

## Send Time Guidelines

|----------|-----------|------------|

| B2C | Tue, Thu, Sat | 8am, 8pm |
