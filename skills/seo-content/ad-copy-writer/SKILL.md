## Purpose

Create compelling advertising copy that captures attention, communicates value, and drives action across digital platforms.

## Quick Reference

### Platform Specifications

| Platform | Headline | Description | Character Limits |
|----------|----------|-------------|------------------|
| Google Search | 30 chars × 3 | 90 chars × 2 | Headlines + descriptions |
| Google Display | 25-30 chars | 80-90 chars | Varies by format |
| Facebook/Meta | 40 chars | 125 chars primary | 27 headline, 27 link |
| LinkedIn | 70 chars | 150 chars intro | 100 headline |
| Twitter/X | N/A | 280 chars | Including link |

### Ad Performance Benchmarks

| Platform | CTR (Good) | CVR (Good) |
|----------|------------|------------|
| Google Search | >3% | >5% |
| Google Display | >0.5% | >1% |
| Facebook | >1% | >2% |
| LinkedIn | >0.5% | >3% |

## Ad Copy Framework (AIDA)

| Stage | Goal | Techniques |
|-------|------|------------|
| **A**ttention | Stop the scroll | Power words, questions, numbers |
| **I**nterest | Build curiosity | Benefits, relevance, story |
| **D**esire | Create want | Social proof, scarcity, emotion |
| **A**ction | Drive click | Clear CTA, urgency |

## Google Ads Templates

### Search Ad Format

```
HEADLINES (3 × 30 chars each):
H1: [Primary benefit/keyword]
H2: [Secondary benefit/feature]
H3: [CTA or trust element]

DESCRIPTIONS (2 × 90 chars each):
D1: [Expand on benefit, include CTA]
D2: [Social proof or secondary benefit]

EXAMPLE:
H1: Save 50% on CRM Software
H2: Free Trial - No Credit Card
H3: Trusted by 10,000+ Companies

D1: Streamline your sales process and close more deals. Start your free 14-day trial today.
D2: Rated #1 by G2 for ease of use. Join thousands of happy customers. Get started now.
```

### Responsive Search Ad

```
HEADLINES (up to 15, 30 chars each):
1. [Primary keyword + benefit]
2. [Different benefit angle]
3. [Question format]
4. [Number-focused]
5. [Social proof]
6. [Urgency element]
7. [Price/offer]
8. [Brand + differentiator]
9-15. [Variations]

DESCRIPTIONS (up to 4, 90 chars each):
1. [Main value prop + CTA]
2. [Feature-benefit + CTA]
3. [Social proof + CTA]
4. [Urgency + CTA]

PIN: H1 for brand consistency
```

### Google Display Ad

```
SHORT HEADLINE (25 chars):
[Benefit or hook]

LONG HEADLINE (90 chars):
[Expanded benefit with context]

DESCRIPTION (90 chars):
[Value prop + CTA]

BUSINESS NAME (25 chars):
[Your brand]

EXAMPLE:
Short: Cut Costs by 40%
Long: See How 500+ Companies Reduced Operating Costs by 40%
Desc: AI-powered expense management. Start free trial.
Name: ExpensifyAI
```

## Facebook/Meta Ads Templates

### Single Image Ad

```
PRIMARY TEXT (125 chars):
[Hook] + [Benefit] + [CTA]

HEADLINE (27 chars):
[Clear offer or benefit]

DESCRIPTION (27 chars):
[Supporting info]

EXAMPLE:
Primary: Tired of wasting hours on reports? Our AI generates them in seconds. Try free →
Headline: Reports in 30 Seconds
Description: Free 14-Day Trial
```

### Long-Form Ad (Primary Text)

```
[HOOK - Stop the scroll, 1 line]

[PROBLEM - Relate to their pain, 2-3 lines]

[SOLUTION - Introduce your product, 2-3 lines]

[PROOF - Social proof or result, 1-2 lines]

[CTA - What to do next]

[EMOJI USAGE - Sparingly to break up text]

EXAMPLE:
🤯 We grew from $0 to $1M in 12 months.

Everyone said e-commerce was "too competitive."
That without a huge budget, we'd never make it.

Then we discovered THIS customer acquisition strategy.

It helped us get customers for 1/3 the cost of our competitors.

Now we're sharing the exact playbook.

👉 Click below to get the free guide.
```

### Carousel Ad

```
CARD 1 (Hook):
Image: [Eye-catching visual]
Headline: [Problem statement or question]
Description: [Tease the solution]

CARD 2-4 (Benefits):
Image: [Relevant visual]
Headline: [Benefit point]
Description: [Brief explanation]

CARD 5 (CTA):
Image: [Product or action visual]
Headline: [Clear CTA]
Description: [Urgency or offer]
```

## LinkedIn Ads Templates

### Sponsored Content

```
INTRO TEXT (150 chars):
[Professional hook + value + CTA]

HEADLINE (100 chars):
[Benefit-focused, professional tone]

EXAMPLE:
Intro: B2B sales teams using AI close 30% more deals. See how leading companies are transforming their pipeline. Read the report.
Headline: The AI Sales Playbook: 2024 Benchmarks & Best Practices
```

### Message Ad

```
SUBJECT (60 chars):
[Personalized + benefit]

BODY (500 chars):
Hi [First Name],

[Personal connection or observation]

[Quick value proposition]

[Clear next step]

[Signature]

EXAMPLE:
Subject: Quick question about [Company]'s sales process

Hi [First Name],

Noticed [Company] is growing quickly - congrats!

At this stage, most sales teams struggle to scale outreach without sacrificing quality.

We've helped 50+ companies in [industry] solve this.

Would you be open to a quick chat?

[Your Name]
```

## Twitter/X Ads

```
TWEET COPY (280 chars including link):
[Hook] + [Value] + [CTA] + [Link]

VARIATIONS:
- Question format
- Statement format
- Social proof format
- How-to format

EXAMPLE:
Question: Still manually creating reports? 🤔
Value: Our AI generates them in 30 seconds.
CTA: Try free for 14 days.
Link: [URL]

"Still manually creating reports? 🤔 Our AI generates them in 30 seconds. Try free for 14 days. [URL]"
```

## Copy Elements

### Headlines That Work

| Type | Example | Use When |
|------|---------|----------|
| How-to | "How to [Result] in [Time]" | Educational |
| Question | "Tired of [Problem]?" | Problem-aware |
| Number | "[X]% Off Today Only" | Promotional |
| Command | "Stop [Pain]. Start [Gain]." | Direct |
| Testimonial | "Best [Product] I've Used" | Social proof |

### CTAs by Goal

| Goal | CTAs |
|------|------|
| Lead gen | Get Free Guide, Download Now, Sign Up |
| Trial | Start Free Trial, Try for Free, Get Started |
| Purchase | Buy Now, Shop Now, Order Today |
| Learn more | Learn More, See How, Discover |
| Contact | Contact Us, Get a Quote, Talk to Sales |

### Urgency Elements

| Type | Example |
|------|---------|
| Time-limited | "Ends midnight" |
| Quantity-limited | "Only 5 spots left" |
| Price-limited | "Price increases Friday" |
| Bonus-limited | "Free shipping this week only" |

### Social Proof Types

| Type | Example |
|------|---------|
| Numbers | "Join 50,000+ customers" |
| Names | "Used by Nike, Airbnb, Spotify" |
| Rating | "Rated 4.9/5 stars" |
| Awards | "Winner: Best Software 2024" |
| Results | "Customers see 3x ROI" |

## A/B Testing Framework

### What to Test

| Element | Priority | Test Ideas |
|---------|----------|------------|
| Headlines | High | Benefit vs. feature, question vs. statement |
| CTA | High | Different verbs, urgency vs. no urgency |
| Social proof | Medium | With vs. without, different types |
| Offer | Medium | Discount vs. free trial |
| Tone | Low | Formal vs. casual |

### Testing Process

```
1. Identify element to test
2. Create 2 variations (1 variable only)
3. Run with equal budget
4. Wait for statistical significance
5. Declare winner
6. Iterate on winner
```

## Platform Best Practices

### Google Ads

- Include keywords in headlines
- Use all available headlines/descriptions
- Pin brand terms if needed
- Include numbers and CTAs
- Use dynamic keyword insertion carefully

### Facebook/Meta Ads

- Front-load key message
- Use emojis strategically
- Short paragraphs
- Mobile-first (80%+ mobile)
- Test video vs. image

### LinkedIn Ads

- Professional tone
- Industry-specific language
- Thought leadership angle
- Personalization when possible
