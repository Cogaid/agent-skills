# Twitter/X Templates

## Single Tweet Templates

### Insight Tweet
```
[Bold statement or insight]

Here's why:

[Brief explanation in 2-3 lines]
```

### List Tweet
```
[X] things I learned about [topic]:

1. [Point]
2. [Point]
3. [Point]
4. [Point]

Which resonates most?
```

### Question Tweet
```
[Thought-provoking question]

I used to think [old belief].

Now I know [new insight].

What's your take?
```

### Hot Take
```
Unpopular opinion:

[Contrarian statement]

Here's what people get wrong:

[Explanation]

Agree or disagree?
```

## Thread Templates

### Lessons Thread
```
TWEET 1 (HOOK):
I [action/experience] for [time period].

Here are [X] lessons that changed everything:

🧵

TWEET 2:
1/ [First lesson]

[Explanation with specific example]

TWEET 3:
2/ [Second lesson]

[Explanation with specific example]

[Continue pattern...]

FINAL TWEET:
That's [X] lessons from [experience].

TL;DR:
• [Point 1]
• [Point 2]
• [Point 3]

Follow @[handle] for more on [topic]

♻️ Retweet if this helped
```

### How-To Thread
```
TWEET 1 (HOOK):
How to [achieve outcome] in [timeframe]:

(No [common obstacle] required)

🧵

TWEET 2:
First, understand this:

[Key context or misconception]

TWEET 3:
Step 1: [Action]

[Why this matters]
[How to do it]

TWEET 4:
Step 2: [Action]

[Why this matters]
[How to do it]

[Continue...]

FINAL TWEET:
Recap:
1. [Step]
2. [Step]
3. [Step]

Save this thread for later.

Follow @[handle] for more [topic] tips.
```

### Story Thread
```
TWEET 1 (HOOK):
[Year], I [dramatic situation].

Here's what happened next:

🧵

TWEET 2:
It started when [context].

[Setup the situation]

TWEET 3:

[What changed]

The result?

[Outcome with specific numbers if possible]

FINAL TWEET:

[Key takeaway]

This changed how I think about [topic].

What's your biggest lesson from [similar experience]?
```

## Best Practices

### Formatting
- One idea per tweet
- Line breaks for readability
- Max 2-3 hashtags per tweet
- Emojis as visual breaks, not decoration

### Thread Tips
- Hook in first 5 words
- Each tweet should stand alone
- End with clear CTA
- Pin best threads

### Timing
- Test different times
- Weekday mornings often work well
- Threads: Early morning or evening
- Avoid holidays/major news days
