## Single Source Repurposing Schedule

```
SOURCE: [Original content title]
URL: [Link]
PUBLISH DATE: [Date]

REPURPOSING TIMELINE
====================

DAY 0 - Source Published
[ ] Blog post live
[ ] Share on all platforms (teaser)

DAY 1 - Primary Distribution
[ ] Twitter thread (main points)
[ ] LinkedIn carousel (visual)
[ ] Email newsletter excerpt

DAY 3 - Secondary Content
[ ] Instagram carousel
[ ] Twitter thread (different angle)
[ ] LinkedIn story post

DAY 5 - Visual Content
[ ] Quote graphic #1 (Instagram)
[ ] Quote graphic #2 (Twitter)
[ ] Quote graphic #3 (LinkedIn)

DAY 7 - Video Content
[ ] TikTok/Reels script written
[ ] Video filmed
[ ] Video posted

DAY 14 - Resurface
[ ] Share best performing version
[ ] Quote tweet original thread
[ ] Respond to old comments

DAY 30 - Refresh
[ ] New angle on same content
[ ] Update if new info available
[ ] Repurpose into new format
```

## Content Batching Template

```
BATCH SESSION: [Date]
TIME ALLOCATED: [X hours]
=====================

CONTENT SOURCES TO REPURPOSE:
1. [Source 1]
2. [Source 2]
3. [Source 3]

SESSION GOALS:
[ ] Create [X] Twitter threads
[ ] Create [X] LinkedIn posts
[ ] Create [X] Instagram carousels
[ ] Create [X] video scripts
[ ] Create [X] quote graphics

WORKFLOW:
---------
Hour 1: Extract elements from all sources
Hour 2: Write Twitter threads
Hour 3: Write LinkedIn posts
Hour 4: Create carousel outlines
Hour 5: Write video scripts
Hour 6: Schedule everything

OUTPUT INVENTORY:
| Type              | Created | Scheduled | Week to Post |
```
