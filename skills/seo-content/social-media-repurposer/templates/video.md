# Short-Form Video Templates

## Script Structures

### The Hook-Value-CTA Structure
```
HOOK (0-3 seconds):
[Attention-grabbing statement or question]
"Stop [doing wrong thing]"
"Here's why [common belief] is wrong"
"[X] [things] you're doing wrong"

VALUE (3-45 seconds):
[Deliver on the hook promise]
- Point 1 with quick example
- Point 2 with quick example
- Point 3 with quick example

CTA (final 5-15 seconds):
"Follow for more [topic]"
"Save this for later"
"Comment if you relate"
```

## Video Type Templates

### Quick Tips Video (15-30 sec)
```
SCRIPT:
---
HOOK (0-3s):
"[X] [topic] hacks you need to know"

POINT 1 (3-10s):
"First, [tip]. This works because [why]."

POINT 2 (10-20s):
"Second, [tip]. Try this [when/where]."

POINT 3 (20-27s):
"Third, [tip]. This is a game-changer."

CTA (27-30s):
"Follow for more tips."

---
ON-SCREEN TEXT:
- Title card with topic
- Point 1 as text overlay
- Point 2 as text overlay
- Point 3 as text overlay
- "Follow @handle"
```

### Myth-Busting Video (30-45 sec)
```
SCRIPT:
---
HOOK (0-3s):
"[Common belief] is completely wrong."

THE MYTH (3-10s):
"Everyone thinks [myth]. I used to think this too."

THE TRUTH (10-35s):
"But here's the reality: [truth]."
"[Evidence or personal experience]"
"This means [implication]."

CTA (35-45s):
"Stop believing myths. Follow for more truth."

---
ON-SCREEN TEXT:
- "MYTH: [belief]" ❌
- "REALITY: [truth]" ✅
- Key stats or evidence
```

### Story-Based Video (45-60 sec)
```
SCRIPT:
---
HOOK (0-3s):
"I [did thing] and [unexpected result]."

CONTEXT (3-15s):
"[Time] ago, I was [situation]."
"I tried [common approach]."

TURNING POINT (15-30s):
"Then [what changed]."
"I discovered [insight]."

RESULT (30-45s):
"Now [outcome]."
"Here's what I learned: [lesson]."

CTA (45-60s):
"Follow for more stories like this."

---
VISUAL SUGGESTIONS:
- B-roll of relevant activities
- Before/after if applicable
- Text overlays for key points
```

### Tutorial Video (45-60 sec)
```
SCRIPT:
---
HOOK (0-3s):
"How to [achieve outcome] in [timeframe]."

STEP 1 (3-15s):
"First, [action]."
[Show demonstration]
"This is important because [why]."

STEP 2 (15-30s):
"Next, [action]."
[Show demonstration]
"Pro tip: [insight]."

STEP 3 (30-45s):
"Finally, [action]."
[Show demonstration]
"And you're done."

CTA (45-60s):
"Save this. Follow for more tutorials."

---
ON-SCREEN TEXT:
- "Step 1: [action]"
- "Step 2: [action]"
- "Step 3: [action]"
```

### POV/Relatable Video (15-30 sec)
```
SCRIPT:
---
ON-SCREEN TEXT:
"POV: [relatable situation]"

ACTION (entire video):
[Act out the scenario]
[Exaggerate for humor/effect]

---
NO VOICEOVER - Text and action only
Add trending sound
```

### Listicle Video (30-60 sec)
```
SCRIPT:
---
HOOK (0-3s):
"[X] [items] every [audience] needs"

ITEM 1 (3-12s):
"Number 1: [item]"
"[Why it's valuable]"

ITEM 2 (12-24s):
"Number 2: [item]"
"[Why it's valuable]"

ITEM 3 (24-36s):
"Number 3: [item]"
"[Why it's valuable]"

[Continue as needed...]

CTA (final 5s):
"Which is your favorite? Comment below."

---
ON-SCREEN TEXT:
- Numbered items
- Product names/links if applicable
```

## Hook Formulas

### Question Hooks
- "Did you know [surprising fact]?"
- "What if I told you [counterintuitive claim]?"
- "Why does no one talk about [overlooked topic]?"

### Challenge Hooks
- "Stop [wrong action] right now"
- "You're [doing X] all wrong"
- "Delete this from your [area] immediately"

### Curiosity Hooks
- "[X] things [group] will never tell you"
- "The [topic] secret that changed my life"
- "This is why you're not [achieving goal]"

### Social Proof Hooks
- "How I went from [before] to [after]"
- "This [approach] got me [impressive result]"
- "[Number] people are doing this wrong"

### Promise Hooks
- "[Outcome] in [timeframe]"
- "[X] steps to [desirable result]"
- "The only [thing] you need for [outcome]"

## Platform-Specific Tips

### TikTok
- Optimal length: 15-30 seconds
- Use trending sounds
- Text on screen essential
- Reply to comments with video
- Post 1-3x daily

### Instagram Reels
- Optimal length: 15-30 seconds
- Can use trending audio or original
- High-quality visuals preferred
- Cross-post from TikTok (remove watermark)
- Post 1-2x daily

### YouTube Shorts
- Optimal length: 30-60 seconds
- Vertical format (9:16)
- Can be more educational/longer
- Great for tutorial content
- Post 1-2x daily

## Production Checklist

```
PRE-PRODUCTION:
[ ] Script written
[ ] Hook tested (first 3 seconds)
[ ] Visual plan/storyboard
[ ] Equipment ready

FILMING:
[ ] Good lighting (natural or ring light)
[ ] Clean background
[ ] Stable camera (tripod recommended)
[ ] Clear audio (test first)

POST-PRODUCTION:
[ ] Cut dead air
[ ] Add text overlays
[ ] Add captions
[ ] Choose audio/music
[ ] Review hook strength

POSTING:
[ ] Write engaging caption
[ ] Add relevant hashtags
[ ] Post at optimal time
[ ] Engage with comments
```

## Content Ideas by Type

### Educational
- [X] tips for [outcome]
- How to [achieve result]
- [Common thing] explained in 30 seconds
- Beginner mistakes in [topic]
- The right way to [do thing]

### Entertainment/Relatable
- POV: [common experience]
- Things only [group] understand
- When [relatable situation]
- Day in the life of [role]
- Expectation vs reality: [topic]

### Inspirational
- My journey from [start] to [now]
- What I wish I knew about [topic]
- [Failure] that led to [success]
- Why I [unconventional choice]

### Behind the Scenes
- How I create [content]
- A day in my life
- My [morning/work/creative] routine
- How this [product/content] was made
