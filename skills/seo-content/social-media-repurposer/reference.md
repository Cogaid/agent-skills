## Content Atomization

Turn one long-form piece into multiple assets:

```
BLOG POST / ARTICLE
        │
        ├── Key insight threads (Twitter/X)
        ├── Carousel breakdowns (LinkedIn, Instagram)
        ├── Quote graphics (Instagram, Twitter)
        ├── Short-form video scripts (TikTok, Reels)
        ├── Discussion posts (LinkedIn, Facebook)
        ├── Poll/Question posts (All platforms)
        ├── List posts (Twitter threads)
        ├── Story series (Instagram, LinkedIn)
        └── Newsletter excerpt (Email)
```

## Platform Guidelines

### Twitter/X

**Character limit**: 280 per tweet

**Thread Format**:
```
Tweet 1: [Hook - create curiosity]

🧵

Tweet 2: [First point]

Tweet 3: [Second point]

Tweet 4: [Third point]

Tweet 5: [Example or proof]

Tweet 6: [Summary + CTA]
```

**Best Practices**:
- Hook in first line
- Line breaks for readability
- End with follow CTA
- Quote tweet to resurface

---

### LinkedIn

**Character limit**: 3,000 (140 visible before fold)

**Story Post Format**:
```
[Hook line that creates curiosity]

I [action that sets up story].

Here's what happened:

[Story with details]

[The lesson]

[Engagement question]

---
#hashtags (3-5)
```

**Carousel Format**:
```
Slide 1: [Hook/Title]
Slide 2: [Problem/Context]
Slides 3-7: [Key points]
Slide 8: [Summary]
Final: [Follow CTA]
```

**Best Practices**:
- Strong hook in first 2 lines
- Personal stories perform well
- Carousels get high engagement
- End with a question

---

### Instagram

**Caption limit**: 2,200 characters

**Carousel Format**:
```
Cover: [Bold title]
Slide 2: [Problem/Hook]
Slides 3-7: [Points with visuals]
Slide 8: [Takeaway]
Final: [Save/Share CTA]
```

**Reel Format** (15-60 seconds):
```
Hook (0-3s): [Attention grabber]
Setup (3-10s): [Context]
Content (10-45s): [Points]
CTA (45-60s): [Follow/Save]
```

**Best Practices**:
- High-quality visuals
- Carousels get saves
- Reels get reach
- Hashtags in caption

---

### TikTok / Reels / Shorts

**Format**: Vertical video (9:16)
**Sweet spot**: 30-45 seconds

**Script Template**:
```
HOOK (0-2s):
"[Question or bold statement]"

CONTEXT (2-8s):
"[Brief setup]"

CONTENT (8-45s):
"Point 1: [Statement]
Point 2: [Statement]
Point 3: [Statement]"

CTA (final 5s):
"[Follow/Comment/Save]"
```

**Video Ideas**:
- "3 things you didn't know..."
- "Stop doing [common mistake]"
- "The easiest way to..."
- "[Topic] explained in 30 seconds"

## Repurposing Templates

### Blog → Twitter Thread

```
Tweet 1:
I analyzed [topic].

Here are [X] lessons:

🧵

Tweet 2:
1/ [First lesson]

[Brief explanation]

Tweet 3:
2/ [Second lesson]

[Brief explanation]

[Continue...]

Final Tweet:
That's it! [X] ways to [outcome].

If helpful:
→ Follow @handle for more
→ Retweet to share

[Link optional]
```

### Blog → LinkedIn Carousel

```
Slide 1 (Cover):
"[Compelling title]"
[Eye-catching design]

Slide 2 (Problem):
"[The common challenge]"
"[Why it matters]"

Slides 3-7 (Content):
Each slide = 1 tip
- Bold headline
- 2-3 bullet points

Slide 8 (Summary):
Recap all points

Slide 9 (CTA):
"Follow for more"
[@handle]
```

### Blog → Quote Graphics

Extract quotable moments:
```
QUOTE 1:
"[Memorable insight]"
- From [section]

QUOTE 2:
"[Actionable advice]"
- From [section]

QUOTE 3:
"[Bold statement]"
- From [section]
```

## Amplification Sequence

```
Day 0: Publish source content
Day 1: Twitter thread + LinkedIn carousel
Day 3: Instagram carousel
Day 5: Quote graphic series
Day 7: Short-form video
Day 14: Resurface best performer
Day 30: Repurpose with new angle
```

## Evergreen Recycling

High performers can be re-shared:
- Same content, new time
- Updated angle/hook
- Different format
- Seasonal relevance
- Response to trending topic
