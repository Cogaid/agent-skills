## On-Page SEO Checklist

### Title Tag
- [ ] Length: 50-60 characters
- [ ] Keyword placement: Front-loaded
- [ ] Unique from other pages
- [ ] Compelling: Encourages clicks

**Formulas**:
```
[Primary Keyword]: [Benefit/Hook] | [Brand]
How to [Action] [Outcome] in [Timeframe]
[Number] [Adjective] Ways to [Achieve Outcome]
```

### Meta Description
- [ ] Length: 150-160 characters
- [ ] Include target keyword
- [ ] Clear value proposition
- [ ] Call to action
- [ ] Unique to page

**Formula**:
```
[Hook/Problem] + [What they'll get] + [CTA]
```

### Header Structure
- [ ] One H1 only (includes keyword)
- [ ] H2s cover main topics
- [ ] H3s break down H2 sections
- [ ] Descriptive, not clever
- [ ] Scannable

### Content Structure
- [ ] Paragraphs: 2-3 sentences max
- [ ] Sentences: Average 15-20 words
- [ ] Bullet points for lists
- [ ] Subheadings every 200-300 words
- [ ] Visuals to break up text

### Keywords
- [ ] Primary in title tag
- [ ] Primary in H1
- [ ] Primary in first 100 words
- [ ] Primary in at least one H2
- [ ] Secondary keywords natural
- [ ] In image alt text
- [ ] No stuffing

### Internal Linking
- [ ] Links TO this page from high-authority pages
- [ ] Links FROM this page to relevant content
- [ ] Descriptive anchor text
- [ ] Contextually relevant

## Content Gap Analysis

### Template

```
CONTENT GAPS
------------
Your Content:
- Word count: [#]
- Topics covered: [List]
- Unique angle: [What's different?]

Top Competitor (average of top 3):
- Word count: [#]
- Topics they cover (you don't): [List]
- Topics you cover (they don't): [List]

GAPS TO FILL:
- [ ] [Missing topic 1] - Add section: [Heading]
- [ ] [Missing topic 2] - Add section: [Heading]

DEPTH IMPROVEMENTS:
- [ ] [Section]: Expand with [what to add]
- [ ] [Section]: Add [data/examples]
```

### Questions to Answer
From "People also ask" and competitor content:
- [ ] [Question 1] - Not answered
- [ ] [Question 2] - Partially answered
- [ ] [Question 3] - Could be deeper

## CTR Optimization

### Title Improvements

**Add numbers**:
- "Ways to..." → "7 Proven Ways to..."

**Add brackets**:
- "Email Tips" → "Email Tips [2025 Guide]"

**Add power words**:
- Ultimate, Proven, Essential, Simple
- Secrets, Mistakes, Hacks
- Fast, Free, New

**Create curiosity**:
- "Why Most People Fail at..."
- "The Mistake Everyone Makes..."

### Meta Description Improvements

**Lead with benefit**:
- What they'll learn/get

**Use action words**:
- Discover, Learn, Master, Unlock

**Include hook**:
- Statistic, question, bold claim

**Add social proof**:
- "Used by 10,000+ marketers"

## E-E-A-T Improvements

### Experience Signals
- [ ] Add first-hand examples
- [ ] Include screenshots of your work
- [ ] Share specific results
- [ ] Add "in my experience" sections

### Expertise Signals
- [ ] Cite authoritative sources
- [ ] Include data/statistics
- [ ] Use correct terminology
- [ ] Add technical depth

### Authoritativeness Signals
- [ ] Add author byline with credentials
- [ ] Link to author page
- [ ] Reference other work

### Trustworthiness Signals
- [ ] Add publication/update dates
- [ ] Cite sources
- [ ] Disclose affiliations
- [ ] Include contact info

## Freshness Updates

### When to Update
- Information is outdated
- Statistics are old
- Links are broken
- Screenshots show old UI
- Year references are past

### Update Process
1. Review all facts and figures
2. Update outdated statistics
3. Replace old screenshots
4. Add new developments
5. Remove irrelevant sections
6. Update publication date
7. Add "Updated: [Date]" notice
