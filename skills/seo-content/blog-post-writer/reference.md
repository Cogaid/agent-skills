```
GOAL:
- Business goal: [Desired action]

COMPETITOR ANALYSIS:
- Top 3 URLs: [List]

```

## Writing Guidelines

### Headlines

**Formulas that work**:
- How to [Action] [Outcome]
- [Number] Ways to [Achieve Result]
- [Number] [Things] Every [Audience] Should Know
- The Complete Guide to [Topic]
- [Option A] vs [Option B]: [Framework]

**Power words**:
- Ultimate, Complete, Essential, Proven, Simple
- Secrets, Mistakes, Hacks, Tips, Strategies
- Fast, Easy, Free, New, Now

### Introductions (First 100 Words)

**Hook formula**:
1. Problem/Pain: Start with reader's challenge
2. Empathy: Show you understand
3. Promise: What they'll get
4. Credibility: Why trust you (brief)

**Example**:
```
You've tried everything to [goal], but [problem persists].

You're not alone. [Statistic or validation.]

In this post, I'll show you exactly how to [promise], based on
[credibility - experience/data].

Here's what we'll cover: [brief outline]
```

### Body Content

**Paragraph rules**:
- 1-3 sentences max
- One idea per paragraph
- Use transition phrases

**Sentence rules**:
- Vary length (10-25 words)
- Active voice preferred
- Remove filler ("very," "really," "just")
- Specific beats vague

**Formatting**:
- Subheadings every 200-300 words
- Bullet lists for series
- Numbered lists for sequences
- Bold for key terms
- Pull quotes for emphasis

### Conclusions

**Formula**:
1. Summarize key takeaways (3-5 bullets)
2. Reinforce main point
3. Inspire action
4. Clear CTA

## SEO Checklist

### Before Writing
- [ ] Target keyword identified
- [ ] Search intent understood
- [ ] Competitor content analyzed
- [ ] Content gap identified

### While Writing
- [ ] Keyword in title (front-loaded)
- [ ] Keyword in first 100 words
- [ ] Keyword in at least one H2
- [ ] Secondary keywords natural
- [ ] Internal links to relevant content
- [ ] External links to authoritative sources

### After Writing
- [ ] Meta title (50-60 characters)
- [ ] Meta description (150-160 characters)
- [ ] URL slug (short, includes keyword)
- [ ] Image alt text includes keyword
- [ ] Featured image selected
- [ ] Categories/tags assigned

### Quality Check
- [ ] Answers search intent completely
- [ ] More valuable than competitors
- [ ] No thin sections
- [ ] Proofread for errors
- [ ] Facts verified
- [ ] Links work

## E-E-A-T Guidelines

### Experience
Show you've actually done this:
- Personal examples and results
- Screenshots and proof
- Specific details only experience provides

### Expertise
Demonstrate knowledge:
- Accurate, up-to-date information
- Nuanced takes, not surface-level
- Original insights

### Authoritativeness
Build credibility:
- Author bylines with credentials
- References to data sources
- Industry recognition

### Trustworthiness
Be reliable:
- Fact-check everything
- Disclose affiliations
- Update content when it ages

## Editing Checklist

### First Pass: Structure
- [ ] Intro hooks the reader?
- [ ] Structure logical?
- [ ] Transitions smooth?
- [ ] Every section adds value?
- [ ] Conclusion satisfying?

### Second Pass: Clarity
- [ ] Every sentence clear?
- [ ] Complex ideas explained simply?
- [ ] Jargon defined or avoided?

### Third Pass: Engagement
- [ ] Interesting to read?
- [ ] Enough examples?
- [ ] Tone consistent?
- [ ] Sounds human?

### Final Pass: Polish
- [ ] Spelling/grammar correct?
- [ ] Formatting consistent?
- [ ] Links working?
- [ ] Images have alt text?
- [ ] CTA clear?
