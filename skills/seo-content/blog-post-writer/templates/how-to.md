## Writing Tips for How-To Posts

### Clarity Guidelines
- Use simple, direct language
- One instruction per step
- Start steps with action verbs
- Include expected outcomes for each step
- Add visuals for complex steps

### Structure Tips
- Number of steps: 5-10 is ideal
- Include time estimates when possible
- Add difficulty indicators
- List prerequisites clearly
- Include a summary/recap

### SEO Considerations
- Title: "How to [Keyword] + [Modifier]"
- H2s should include action words
- Include "how to" in first 100 words
- Answer related "People Also Ask" questions
- Add schema markup for How-To
